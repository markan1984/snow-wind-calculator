Определение снеговых нагрузок на многопролетные покрытия

Сбор снеговой нагрузки ведем согласно указаниям СН 2.01.04.

Согласно указаниям национального приложения НП.1 на территории РБ применимы только нормальные условия, установленные в соответствии с 3.2(1).

Характеристическое значение снеговой нагрузки на двускатное покрытие для постоянных/переходных расчетных ситуаций следует определять согласно п.5.2(3):

s=μi∙Ce∙Ct∙sk

Где:

  • μ1(α1)={{ mui1 }} - коэффициент формы снеговых нагрузок для первого ската кровли при α1={{ slope1 }}° и с учетом п.5.3.4(2);
  • μ1a(α2)={{ mui2a }} - коэффициент формы снеговых нагрузок для второго ската кровли при α2={{ slope2 }}° и с учетом соседнего ската;
  • μ1a(α1)={{ mui1a }} - коэффициент формы снеговых нагрузок для первого ската кровли при α1={{ slope1 }}° и с учетом соседнего ската;
  • μ1(α2)={{ mui2 }} - коэффициент формы снеговых нагрузок для второго ската кровли при α2={{ slope2 }}° и с учетом п.5.3.4(2);
  • μ2(α)={{ mu2 }} - коэффициент формы снеговых нагрузок для центральной части кровли с учетом заноса снега при α=(α1+α2)/2 и с учетом табл 5.2 и п.5.3.4(3);
  • Сe={{ Ce }} - коэффициент окружающей среды (табл. НП.1.3(BY));
  • Сt={{ Ct }} - температурный коэффициент (п.5.2(8) НП.1(BY));
  • sk={{ Sk_formula }}{{ Sk }}кПа - характеристическое значение снеговых нагрузок на грунт для района г.{{ city }} (район {{ snow_area }}) при абсолютной отметке местности А={{ level }};
